#ehour-report

## Description

Basic script that sends activity iby e-mail from eHour database.

See http://www.ehour.nl for more information.

## Usage

    $ ./ehour-report.py -c ehour-report.cfg

## Requirements

* a working e-mail server
* Python >= 2.6

### Python modules

* psycopg2
* smtplib
